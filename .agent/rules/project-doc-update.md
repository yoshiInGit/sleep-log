# 指示

以下のタスクを実行せよ

## 日付を更新する

プロジェクト関連のファイルを更新する場合、そのファイルを最終更新日を今日の日付に更新する。

## 変更履歴を残す

プロジェクト関連のファイルを更新する場合、何を変更したかを簡単にファイルの末尾に残す。

### 例

```
## 変更履歴
- 26/02/10: ○○を変更
```

# 制約

更新しなかったファイルについては、確認も更新も何も行わない。
必ず、編集を適用するファイルのみ確認・更新すること
